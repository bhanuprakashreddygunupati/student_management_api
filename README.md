# Inventory & Order Management SaaS

Engineered an enterprise-grade full-stack solution designed to automate retail supply chains, effectively reducing manual procurement effort by 40%. The system leverages **Spring Boot** and **React 19** to provide a secure, multi-tenant environment with strict Role-Based Access Control (RBAC).

### Key Features
*   **Auto-Reorder Bot:** An intelligent background service that autonomously monitors stock thresholds 24/7 and triggers purchase orders to prevent revenue loss from stockouts.
*   **Data Integrity:** Implemented JPA **Pessimistic Locking** to ensure 100% inventory accuracy during high-concurrency stock adjustments.
*   **Real-Time Analytics:** Interactive Executive Dashboard transforming raw transactional data into instant visual insights on revenue trends and supplier performance.

### Tech Stack
*   **Backend:** Java, Spring Boot 3.5, Hibernate, MySQL, JWT Security.
*   **Frontend:** React 19, Vite, Tailwind CSS, TanStack Query, Recharts.
